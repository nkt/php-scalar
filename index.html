PHP scalar

Methods for PHP scalars.

Installation

Install great @nikic extension that allows add methods to scalar objects.

git clone git://github.com/nikic/scalar_objects.git
cd scalar_objects
phpize
./configure
make
sudo make install

Than add nkt/scalar into your composer.json. That's it!
